# Wave Studio 🌊

Estudio de diseño web futurista y minimalista enfocado en convertir visitantes en clientes.

## 🚀 Características

- **Diseño Futurista:** Estética moderna con glassmorphism y gradientes dinámicos.
- **Optimizado para Conversión:** Estructura pensada para guiar al usuario hacia el contacto.
- **Responsive:** Totalmente adaptable a móviles, tablets y desktop.
- **Animaciones Fluidas:** Uso de Framer Motion para una experiencia premium.
- **Contacto Directo:** Integración con WhatsApp para leads inmediatos.

## 🛠️ Tecnologías

- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **Framer Motion**
- **Lucide Icons**

## 📦 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/LucianoMR5/Wave-studio.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
